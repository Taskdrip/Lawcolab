Security Notice

All API keys and sensitive configuration data are encrypted before storage using Fernet encryption. Keys are never stored in plain text.

Gateway Features

  • Bank Transfer: Wire transfers, manual verification
Testing Mode

When test mode is enabled, only test API keys will be used and no real payments will be processed. Always test before going live!
